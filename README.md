# lipdnet_flask

This app performs functions for the lipd.net website including lipd to noaa conversion and file merging.

## Use

The virtual environment was built with pip using the env_config.txt file "pip install --no-cache-dir -r env_config.txt" underthe virtual environment "pyenv activate lipdnetflask" (python 3.8.0).

If the environmnet is built correctly the app is launched with "python -m flask run" or "nohup python -m flask run > log.txt 2>&1 &"
